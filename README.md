# SnapPlayerToNearest90
Snaps the player to the nearest mutliple of 90 degrees, much like player rotation behavior in older versions of GD.

Note that this mod has a small chance of breaking your favorite macro bot as this mod changes the player's rotational hitbox.

Mod idea suggested by [MolloPro](https://www.youtube.com/channel/UCZ-Sm6ikSOm9MfpqfrF5OZQ) and previously attempted by [AntiMatter](https://github.com/A-Zalt).
